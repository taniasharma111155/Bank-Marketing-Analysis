# Bank Marketing Campaign Analysis

## Project Overview
This project analyzes customer data from a bank marketing campaign to identify factors influencing term deposit subscriptions.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning

## Key Insights
- Identified customer segments with higher subscription rates.
- Analyzed age, occupation, education, and campaign outcomes.
- Built an interactive dashboard for business reporting.
